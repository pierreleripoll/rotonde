HOW TO USE THE WEBSITE



Session Client



Voir les spectacles à l’affiche


  • Depuis la page d’accueil, par spectacle

  • Depuis le calendrier, par dates


Obtenir des informations supplémentaires sur un spectacle


  • Depuis l'accueil
    • Vignette du spectacle

  • Depuis le calendrier, par dates
    • Lien vers la page spectacle


Réserver une place

  • Depuis la page spectacle
    • Rubrique: Billets/Dates
    • Affichage des dates disponibles: sélection des places souhaitées
    • Bouton “Valider”
  • Pour valider la commande
    • Onglet “Panier”
    • Possibilité de modifier la quantité de places souhaitées
    • Saisie du nom et de l’adresse mail
    • Bouton “Valider”
  • Sur place
    • Validation de la place par la billetterie grâce à l'identifiant de la place envoyé par mail
  • Contacter/Trouver la Rotonde
    • Onglet "Contacts"


Session Admin



Se logger


  • Onglet “Admin”
    • Identification par login et mot de passe


Gestion des spectacles


  • Créer un spectacle
    • Bouton “Créer spectacle”

  • Modifier un spectacle
    • Depuis l'accueil comme depuis la page du spectacle
      • Bouton "Edit"
    • Depuis la page spectacle
      • Panneau Billets/Dates
        • Bouton "Modifier"
    • Depuis la page admin
      • Bouton “Modifier” correspondant au spectacle

    • Saisie des informations (DANS L'ORDRE DES OPÉRATIONS)
      • Ajouter une image
        • Bouton “Browse” ou (Drag & Drop)
        • Sélection du fichier
      • Charger une image
        • Bouton “Upload”
      • Modifier une image
        • Bouton "Recadrer"
      • Supprimer une image
        • Bouton "supprimer"
      • Ajouter un contact
        • Choix dans la liste des contacts existants
        • Ou création d'un nouveau contact
          • Bouton "Ajouter Contact"
      • Ajouter une date
        • Bouton "Add"
          • Sélection de la date
    • Supprimer un spectacle
      • Depuis la page spectalce
        • Bouton "Supprimer"
      • Depuis la page de modification du spectacle
        • Bouton "Supprimer"
    • Voir la liste des spectacles de l'admin
      • Onglet Admin
    • Voir les places réservées pour un spectacle
      • Depuis la page Admin
        • Bouton "Voir les réservations" correspondant au spectacle
        • Possibilité de cocher la place correspondante


Gestion des sessions


  • Modifier le compte Admin
    • Depuis la page admin
      • Bouton "Infos du compte"
        • Modification des données(login, mot de passe)
        • Ajout de contacts IDEM que pour CRÉER SPECTACLE
  • Se déconnecter
    • Depuis la page admin
      • Bouton "se déconnecter"


Session Superadmin


  • Voir la liste des admins
    • Depuis la page Admin
      • Bouton "Liste des admins"
  • Ajouter un admin
    • Depuis la page admin
      • Bouton "Ajouter un admin"
        • Saisie des données de l'admin et de son statut
      • Ajout de contacts IDEM que pour CRÉER SPECTACLE
  • Modifier le compte d'un admin
    • Depuis la liste d'admins
      • Bouton "Modifier"
